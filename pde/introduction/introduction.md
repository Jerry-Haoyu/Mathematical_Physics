# Definition

:::{prf:definition} 
A PDE for a function $u(x_1,...,x_n)$ is of the form:
$$
F(x_1,...,x_n, u,u_{x_1},...,u_{x_n},u_{x_1x_1},u_{x_1x_2},...)=0
$$
1. The *order* is the order of the highest-order derivative 
2. An equation is called *linear* if it is linear on $u$ and its derivatives
3. An equation is callled *semi-linear* if all coefficinets are only functions of $x_1,...,x_n$
4. An equation is called *quasilinear* if coefficinet $a_N$ is only functions of $x,u$ and lower-order derivatives.
:::
